Panduan Sinkronisasi Google Sheets

Hubungkan interface web app buku induk ini dengan database awan milik Anda.

Web App ini dirancang siap di-deploy ke dalam Google Apps Script agar terintegrasi langsung dengan database Google Sheets Anda secara real-time. Anda hanya butuh melakukan 3 langkah sederhana:

1 Buat Google Sheets Baru

Buat Google Sheets dan beri nama tab pertama dengan nama "BukuInduk".

2 Buka Apps Script & Paste Kode ini

Dari Google Sheets, klik menu Ekstensi > Apps Script. Hapus kode default bawaan dan ganti dengan kode Code.gs berikut:

// Nama sheet yang akan digunakan sebagai database
const SHEET_NAME = "BukuInduk";

// Fungsi untuk menangani permintaan GET (Ambil Data)
function doGet(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME);
  
  let data = {
    students: sheet.getRange("A2").getValue() || "[]",
    letters: sheet.getRange("B2").getValue() || "[]"
  };
  
  return ContentService.createTextOutput(JSON.stringify({ status: 'success', ...data }))
    .setMimeType(ContentService.MimeType.JSON);
}

// Fungsi untuk menangani permintaan POST (Simpan Data)
function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME);
  
  // Setup Header Kolom jika belum ada
  sheet.getRange("A1").setValue("DATABASE_SISWA_JSON").setFontWeight("bold");
  sheet.getRange("B1").setValue("DATABASE_SURAT_JSON").setFontWeight("bold");
  
  try {
    const parsedData = JSON.parse(e.postData.contents);
    
    if (parsedData.type === 'students') {
       sheet.getRange("A2").setValue(JSON.stringify(parsedData.data));
    } else if (parsedData.type === 'letters') {
       sheet.getRange("B2").setValue(JSON.stringify(parsedData.data));
    }
    
    return ContentService.createTextOutput(JSON.stringify({ status: 'success' }))
      .setMimeType(ContentService.MimeType.JSON);
      
  } catch(error) {
    return ContentService.createTextOutput(JSON.stringify({ status: 'error', message: error.message }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

3 Deploy & Masukkan Link Web App

Klik tombol biru Terapkan (Deploy) > Deployment baru. Pilih tipe Aplikasi Web. Atur "Siapa yang memiliki akses" menjadi Siapa saja (Anyone). Salin link URL Web App yang dihasilkan dan tempel di bawah ini:
